Project Gallery

Steganography

The art of hiding messages within images. This project was made using Kotlin and lets the user store a secret message in an image. The message is also encrypted with a password, and can only be retrieved with the correct password. For detailed documentation and code sample, follow the link here.

Text Editor

A notepad-like text editor that can be used to find occurrences of a particular word in the document. Created using Java and Swing. Supported file formats include, .txt, .csv, .rtf, .json, .md, etc. The GitHub repository contains images showing the basic functionality of the app.

Path Finding

A simple application of path finding that involves generation of a random maze of given dimensions and finds the escape using Prim's Algorithm. Generated maze can be saved to a file. Take a look at the implementation here .
